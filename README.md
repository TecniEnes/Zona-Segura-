# Zona-Segura-
El objetivo principal del proyecto es poder segmentar Michoacán en base al índice de violencia de cada colonia y municipio usando las bases de datos y APIS del gobierno mexicano y fiscalía de Michoacán.

The main objective of the project is to segment Michoacán based on the violence index of each neighborhood and municipality using databases and APIs from the Mexican government and the Michoacán State Prosecutor’s Office.
